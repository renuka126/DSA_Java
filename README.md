# DSA_Java

This is where I'm building up my Data Structures and Algorithms practice in Java, one topic at a time.

## What this is

I started this repo as part of my prep for technical interviews and placements. Instead of solving random problems here and there, I wanted a space where I could actually track my progress topic by topic — so when I look back, I can see how far I've come, and honestly, so I stay a bit more accountable about practicing regularly.

## How it's organized

- `Arrays/` — searching, sorting, two-pointer stuff, sliding window
- `Strings/` — pattern matching, manipulation, palindromes
- `LinkedLists/` — the usual suspects: reversal, cycle detection, basic operations
- `Stacks_Queues/` — implementation plus problems that actually use them
- `Recursion_Backtracking/` — still building intuition here, learning as I go
- `Trees/` — binary trees, BST, traversals
- `Graphs/` — BFS, DFS, and other traversal-based problems
- `DP/` — dynamic programming, working through memoization and tabulation
- `Sorting_Searching/` — writing my own implementations, not just using built-ins

## Why Java

Java's been my go-to language since college, and I've used it enough that I don't have to think about syntax — I can just focus on the logic of the problem instead of fighting the language. It's also the language most companies test candidates in during placement drives here, so it made sense to get really solid in DSA using it before spreading myself across others.

## Where this is headed

This repo is a work in progress and will keep growing as I solve more problems and work through new topics. Not everything is filled in yet — think of this as my roadmap as much as a portfolio.
